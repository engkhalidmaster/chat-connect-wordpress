إضافة احترافية لعرض زر WhatsApp مع تتبع Google Analytics ولوحة تحكم شاملة.

== Description ==

WhatsApp Widget Pro هي إضافة احترافية تتيح لك إضافة زر WhatsApp تفاعلي إلى موقعك الإلكتروني مع ميزات متقدمة:

**الميزات الرئيسية:**

* **لوحة تحكم شاملة** - إدارة جميع الإعدادات من مكان واحد
* **إدارة الفريق** - إضافة متعدد أعضاء خدمة العملاء
* **تخصيص المظهر** - تحكم كامل في شكل وموقع الزر
* **تتبع Google Analytics** - إحصائيات مفصلة للنقرات والمحادثات
* **إحصائيات مفصلة** - تقارير الاستخدام لآخر 30 يوم
* **دعم كامل للعربية** - واجهة باللغة العربية مع دعم RTL
* **تصميم متجاوب** - يعمل بشكل مثالي على جميع الأجهزة

**ميزات الفريق:**

* إضافة أعضاء فريق متعددين
* تحديد حالة كل عضو (متصل/غائب/غير متصل)
* تنظيم الأعضاء حسب الأقسام
* صور شخصية مخصصة لكل عضو

**ميزات المظهر:**

* اختيار موقع الزر (4 مواقع مختلفة)
* تخصيص لون الزر
* رسالة ترحيب قابلة للتخصيص
* تأثيرات حركية جذابة

**ميزات التتبع:**

* ربط مع Google Analytics
* تسجيل النقرات والمحادثات
* إحصائيات يومية مفصلة
* تقارير شاملة للاستخدام

== Installation ==

1. ارفع ملفات الإضافة إلى مجلد `/wp-content/plugins/whatsapp-widget-pro/`
2. فعّل الإضافة من خلال قائمة 'الإضافات' في ووردبريس
3. انتقل إلى 'WhatsApp Widget' في قائمة الإدارة
4. اضبط الإعدادات حسب احتياجاتك
5. أضف أعضاء فريق خدمة العملاء
6. احفظ الإعدادات واستمتع!

== Frequently Asked Questions ==

= هل الإضافة مجانية؟ =

نعم، الإضافة مجانية بالكامل مع جميع الميزات.

= هل تدعم الإضافة اللغة العربية؟ =

نعم، الإضافة مصممة خصيصاً للمواقع العربية مع دعم كامل لـ RTL.

= كيف أربط الإضافة مع Google Analytics؟ =

انتقل إلى تبويب "إعدادات Google Analytics" وأدخل معرف التتبع الخاص بك.

= هل يمكنني إضافة أكثر من شخص للدردشة؟ =

نعم، يمكنك إضافة عدد غير محدود من أعضاء الفريق.

= هل الإضافة متوافقة مع جميع القوالب؟ =

نعم، الإضافة مصممة للعمل مع جميع قوالب ووردبريس.
